# Chrext Chrome Extension

Browser extension for automating repetitive web interactions with macro recording, playback, and scheduled execution.

## Features

- Macro recording for click, type, and navigation actions
- Playback with configurable speed and loop count
- Scheduled task execution
- Element selector builder
- Export and import automation scripts

## Tech Stack

JavaScript, Chrome Extension APIs, HTML, CSS

## Getting Started

### Prerequisites

- Node.js 18+
- npm

### Installation

```bash
git clone https://github.com/karthik-idikuda/Chrext-Chrome-Extension.git
cd Chrext-Chrome-Extension
Load as unpacked extension in chrome://extensions/
```

### Usage

```bash
Click the Chrext icon in the browser toolbar to open the control panel.
```
